# Roadmap

This document outlines the planned direction of the project. Priorities may change as the project evolves.

## Current Status

* Project is under active development.

---

## Short-Term Goals

* Complete core functionality.
* Improve code quality and documentation.
* Increase unit test coverage.
* Optimize build configuration.
* Resolve known issues.

---

## Mid-Term Goals

* Add new coding problems.
* Each and every code should be optimised in terms of time and space complexity rather than brute force approach.
* Enhance error handling and logging.
* Improve developer experience.

---

## Long-Term Goals

* Should be able to use for daily code practices and helpful in the context of Interview preparation.
* Strengthen security and reliability.
* Expand automated testing.
* Prepare stable releases.

---

## Future Ideas

* Plugin or extension support.
* Configuration improvements.
* Additional integrations.
* Performance benchmarking tools.
* Enhanced monitoring and diagnostics.

---

## Out of Scope

The following items are not currently planned:

* Features that significantly increase project complexity without clear value.
* Support for obsolete or unsupported platforms.
* Proprietary or closed-source dependencies.

---

## Contributing

Suggestions, feature requests, and discussions are welcome. Significant changes should be discussed before implementation.
